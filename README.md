# Deck of Cards App 
Простое приложение для работы с колодой карт на Svelte. 
## Функции 
Создание новой колоды 
Вытягивание карт 
Перетасовка колоды 
История вытянутых карт 
Очистка истории 
## Установка 
### bash 
npm install 
npm run dev 
Приложение откроется на http://localhost:5173 
## API 
Использует Deck of Cards API https://deckofcardsapi.com/
